# **App Name**: ASYNCONV SIT

## Core Features:

- Table Display: Display available game tables with game name, time slot, and number of available seats.
- Table Registration: Allow users to select a game table and register, respecting the ticket priority (Strategist, Marshal, General).
- User Schedule: Display the user's registered tables in a clear schedule format.

## Style Guidelines:

- Primary color: Deep green (#2E7D32) to evoke strategy and nature.
- Secondary color: Light beige (#F5F5DC) to provide contrast and readability.
- Accent: Gold (#FFD700) to highlight important elements like available slots or priority status.
- Clean and readable sans-serif font for the main content.
- Use icons to represent game types and available slots.
- Clear and tabular layout for displaying table information.

## Original User Request:
Context: I'm organizing an event, a 4-day convention of history and strategy games.
Each player can choose between 3 types of ticket depending on the length of the stay: Strategist (4 days), Marshal (3 days) or General (2 days).

Table registration system (TRS)
Once the program has been fully unveiled at the end of June, a ticket gives access to the table registration system. Everyone can then register for the tables, and therefore the games, they wish to play. The rank allows first strategists, then marshals and finally generals to register.
So you can start playing at 9:00 on Thursday morning!

It's this table registration system that I want to implement.
Note that the tickets are manage on https://www.billetweb.fr/
